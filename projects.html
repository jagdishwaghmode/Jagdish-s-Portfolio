Disaster Management System

SQL, Python, HTML, CSS

The Disaster Management System is a software-based web application that enables users to report disasters and facilitates quick response by assigning help to affected regions. It allows real-time reporting, maintains databases of available hospital beds, ration shop resources, and volunteers, and helps authorities coordinate relief efforts efficiently. The system ensures that rescue operations are streamlined by providing accurate information and resource allocation through an intuitive user interface.

Digital SchoolPay: Eliminating Cash Dependency

React, NextJS, TypeScript, HTML/CSS

This project is a modern web application developed using React, Next.js, and TypeScript to streamline and digitize the payment process in schools. It integrates the Razorpay payment gateway to enable secure online transactions, eliminating the need for cash handling. The system simplifies fee collection, improves transparency, and provides a user-friendly interface for students, parents, and school administrators.
